# Black Economic Simulator

AI-powered economic simulator for African American business and media wealth building.

## Overview

Black Economic Simulator is a React-based application that models business and media wealth-building strategies for African American entrepreneurs and organizations. It includes an AI business advisor component, interactive charts (Recharts), and tools to simulate investments, revenue growth, hiring, and media engagement strategies.

## Features

- Scenario-based economic simulation for small businesses and media initiatives
- AI-driven business advisor UI component
- Interactive charts and visualizations using Recharts
- Extensible components for policy, investments, and community wealth models

## Getting started

1. Clone the repository:
   ```
   git clone https://github.com/godzillanwk-cmd/black-economic-simulator.git
   cd black-economic-simulator
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Start the development server:
   ```
   npm start
   ```

4. Build for production:
   ```
   npm run build
   ```

## Project structure (high-level)

- src/
  - components/ — React components (includes AIBusinessAdvisor)
  - App.js — app entry
  - App.css
- public/ — static assets
